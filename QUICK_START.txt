🚀 QUICK START (3 STEPS):

1. EXTRACT & SETUP
   $ unzip material-stock-monitoring.zip
   $ cd material-stock-monitoring
   $ cp .env.backend backend/.env
   $ cp .env.frontend frontend/.env.local

2. CONFIGURE
   Edit backend/.env and frontend/.env.local:
   - Set database credentials
   - Set JWT secret
   - Set API URLs

3. RUN WITH DOCKER
   $ bash quickstart.sh

   OR manually:
   $ docker-compose build
   $ docker-compose up -d
